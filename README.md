# Mermaid Viewer

Aplicacion hecha con React + TypeScript para editar y visualizar diagramas Mermaid.

## Scripts

- `npm run dev`
- `npm run start`
- `node server.mjs`

## Incluye

- Editor Mermaid con vista previa en vivo
- Ejemplos cargables
- Manejo de errores de sintaxis
- Exportacion del diagrama como SVG

## Nota

Esta version no requiere instalar dependencias locales. El servidor corre con `node`
y el navegador carga React, Mermaid y el compilador TSX desde CDN.

Abre `http://127.0.0.1:4173` en Chromium despues de iniciar el servidor.
